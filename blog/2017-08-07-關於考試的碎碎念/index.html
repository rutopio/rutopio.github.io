關於「考試」的碎碎念
  • Aug 7, 2017
  • One minute read

其實小時候的我是超級厭世又很討厭考試這種東西的，總覺得考試是一種超就麻煩又不公平的東西，憑甚麼因為他考的比我高就可以拿到比較好的名次和資源，我明明能力也沒有他差啊嗚嗚。

結果越長越大，才漸漸覺得考試是世界上最公平的制度。

就拿指考為例，只要下能夠定決心考指考，你就再也不必擔心你有沒有那個財力和時間去參加一些白痴活動或是白痴營隊來讓自己在學測的備審看起來漂漂亮亮的，也不用擔心在你前面一個面試的同學他老爸是不是這個系的系主任甚至是院長，你只要專心下來唸書，睡覺，然後唸書就夠了。

縱使每間學校都說自己超公平，絕對不會有黑箱作業的情況發生，但是誰知道真的在後面的狀況又怎麼樣呢？黑箱作業如果被知道了還叫黑箱作業逆？

但是如果是純粹的考試，扣除考運超爛、可能年年犯沖的衰人、天降大任於斯人也的感冒、低能出題者出一些意味不明的題目之外，其實你還是掌握者大多數的變因的：沒看完？你的問題；沒看到？你的問題；沒寫完？你的問題；畫錯卡？你的問題。

或許在很多很多年之後，到了那個充滿「大人的原因」的年紀和社會，我會開始懷念我以前考過的所有考試，懷念他們帶來的單純和公平，或許我現在就在懷念了也說不定。
